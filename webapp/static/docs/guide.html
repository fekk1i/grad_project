EZ-Clone Operations Panel User Guide

Overview

The EZ-Clone Operations Panel is a web-based application designed for NFC card management using a Raspberry Pi equipped with a PN532 NFC module. This tool allows users to read, write, and scan NFC tags, particularly focusing on MIFARE Classic 1K cards.

Features

  • Start MFOC Reading: Initiate a reading session to dump the contents of a MIFARE Classic 1K card.
  • Execute NFC Write: Write data to a MIFARE Classic 1K card using previously saved dump files.
  • Perform NFC Scan: Scan for nearby NFC tags to check if they are MIFARE Classic 1K and retrieve their UID (Unique Identifier).
  • Manage .dmp Files: View and refresh the list of available dump files stored on the system.

Getting Started

1. Access the Application

Navigate to the web address provided by the system administrator, typically something like http://raspberrypi.local or an IP address, from any standard web browser.

2. Read a Card

Enter a filename in the "Filename for MFOC Output" field and click "Start MFOC Reading" to dump the data of an inserted NFC card into the specified file. The output will be displayed in the "Live Output Console" area.

3. Write to a Card

To write to a card, both the source (previously dumped) and target (blank card) must have corresponding dump files. Select the appropriate source and target files from the dropdown menus, then click "Execute NFC Write" to begin the process. The progress and results will appear in the "Live Output Console".

4. Scan for NFC Tags

Click on "Perform NFC Scan" to search for nearby NFC tags. The scanning results, including the type of card and its UID, will be displayed in the "Live Output Console".

Troubleshooting & Support

If you encounter issues or need help, check the "Live Output Console" for any error messages, ensure the NFC reader is properly connected and the Raspberry Pi is powered on. For persistent issues, contact the system administrator or technical support team.
